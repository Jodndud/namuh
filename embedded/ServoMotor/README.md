# ESP32 Servo (Serial control)

이 예제는 ESP32에서 서보모터를 직렬 시리얼 명령으로 제어하는 간단한 스케치입니다.

## 폴더 구조

- `ESP32_Servo_Serial/ESP32_Servo_Serial.ino`: 메인 스케치

## 하드웨어 연결

- 서보 신호(S): ESP32 GPIO 18 (기본)
- 서보 VCC: 5V (권장 별도 전원, GND 공통)
- 서보 GND: ESP32 GND와 공통 연결

주의: 서보 전원은 ESP32 보드의 5V에서 직접 끌어쓰지 말고, 별도의 안정적인 5V 공급원을 사용하세요. GND는 반드시 공통으로 묶어야 합니다.

## 의존성

- (선택) 라이브러리 매니저에서 `ESP32Servo` 설치 시 더 간단히 제어됩니다.
  - 없을 경우 스케치는 자동으로 ESP32의 `ledc`(하드웨어 PWM)로 동작합니다.

## 빌드

- Arduino IDE에서 보드: `ESP32 Dev Module`
- 포트 선택 후 업로드

## 시리얼 명령

- 시리얼 모니터 115200 baud, `\n` 전송 설정
- 명령:
  - `0..180` 각도로 이동 (예: `90`)
  - `us=500..2500` 펄스폭 설정 (예: `us=1500`)
  - `sweep` 0→180→0 왕복 시험
  - `help` 도움말 출력

## 설정 값 변경

- 핀/펄스폭/주파수는 스케치 상단의 매크로로 변경 가능
  - `SERVO_PIN`, `SERVO_MIN_US`, `SERVO_MAX_US`, `SERVO_FREQ`

## 트러블슈팅

- 서보가 떨리거나 움직이지 않으면:
  - 별도 5V 전원의 전류 용량 확인 (>= 1A 권장)
  - GND 공통 연결 확인
  - SERVO_PIN이 PWM 가능 핀인지 확인 (예: 18, 19 등)
  - 각도/펄스폭 범위를 줄여 테스트 (예: 600~2400us)
