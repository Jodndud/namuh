멀티 에이전트 시스템은 여러 개의 개별 에이전트(BaseAgent)를 조합하여 더 크고 복잡한 목표를 달성하는 애플리케이션을 의미한다.
이러한 구조는 Agent Development Kit(ADK) 또는 LangGraph를 통해 구현되며, 계층적 구조를 형성하는 에이전트들이 서로 협력하고 조정하며 작동한다.
시스템을 이렇게 구성하면 모듈성, 전문성, 재사용성, 유지보수성이 향상되는 장점이 있다.

핵심적인 개념은 '워크플로우 에이전트'가 오케스트레이터 역할을 수행하며 전체적인 제어 흐름을 정의하는 것이다.
또한, 에이전트 간의 상호작용과 통신을 위해 공유 세션 상태, LLM 기반 위임, 명시적 호출과 같은 메커니즘이 사용된다.
이처럼 ADK는 멀티 에이전트 시스템을 효과적으로 구축하고 관리하는 데 필요한 다양한 기본 기능과 일반적인 패턴을 제공한다.

- 멀티 에이전트 시스템 Python 예시
시스템은 아래와 같이 구성된다.
`ResearchAgent`: 특정 주제에 대한 데이터를 검색하고 수집하는 역할
`SummaryAgent`: 수집된 데이터를 요약하고 핵심 내용을 추출
`ReportWriterAgent`: 요약된 내용을 바탕으로 최종 보고서 초안을 작성
`OrchestratorAgent`: 전체 작업 흐름을 관리합니다. 사용자의 요청을 받아 각 전문 에이전트에게 순서대로 작업을 지시하고, 최종 결과를 반환하는 오케스트레이터 역할을 수행

```Python
# --- 1. 기본 에이전트 클래스 정의 ---
class BaseAgent:
    """모든 에이전트의 기본이 되는 클래스"""
    def run(self, task_description: str) -> str:
        raise NotImplementedError("이 메소드는 서브클래스에서 구현되어야 합니다.")

# --- 2. 전문 분야를 가진 하위 에이전트 정의 ---
class ResearchAgent(BaseAgent):
    """주어진 주제에 대한 정보를 검색하는 에이전트"""
    def run(self, topic: str) -> str:
        print(f"▶ ResearchAgent: '{topic}'에 대한 데이터 검색 중...")
        # 실제로는 웹 검색이나 데이터베이스 조회를 수행합니다.
        return f"'{topic}'에 대한 상세 자료: 멀티 에이전트 시스템은 여러 에이전트가 협력하여 복잡한 문제를 해결하는 패러다임입니다. 각 에이전트는 특정 역할에 전문화되어 있습니다."

class SummaryAgent(BaseAgent):
    """입력된 텍스트를 요약하는 에이전트"""
    def run(self, text_to_summarize: str) -> str:
        print("▶ SummaryAgent: 수집된 데이터 요약 중...")
        # 실제로는 자연어 처리 모델을 사용하여 요약을 수행합니다.
        summary = text_to_summarize.split("입니다.")[0] + "입니다."
        return summary.replace("상세 자료:", "핵심 요약:")

class ReportWriterAgent(BaseAgent):
    """요약된 내용을 바탕으로 보고서를 작성하는 에이전트"""
    def run(self, summary: str) -> str:
        print("▶ ReportWriterAgent: 최종 보고서 작성 중...")
        report = f"""
## 보고서: 멀티 에이전트 시스템 개요

### 1. 핵심 내용
{summary}

### 2. 결론
이 시스템은 모듈성, 재사용성, 유지보수성을 높이는 데 기여합니다.
"""
        return report

# --- 3. 전체 워크플로우를 관리하는 오케스트레이터 에이전트 ---
class OrchestratorAgent(BaseAgent):
    """여러 에이전트의 작업을 조율하여 전체 프로세스를 관리"""
    def __init__(self):
        self.research_agent = ResearchAgent()
        self.summary_agent = SummaryAgent()
        self.report_writer_agent = ReportWriterAgent()

    def run(self, user_request: str) -> str:
        print(f"\n--- OrchestratorAgent: '{user_request}' 작업 시작 ---")

        # 1단계: ResearchAgent를 호출하여 데이터 수집
        research_result = self.research_agent.run(user_request)
        print("--- 1단계 완료: 데이터 수집 완료 ---")

        # 2단계: SummaryAgent를 호출하여 결과 요약
        summary_result = self.summary_agent.run(research_result)
        print("--- 2단계 완료: 데이터 요약 완료 ---")

        # 3단계: ReportWriterAgent를 호출하여 최종 보고서 작성
        final_report = self.report_writer_agent.run(summary_result)
        print("--- 3단계 완료: 보고서 작성 완료 ---\n")

        return final_report

# --- 4. 시스템 실행 ---
if __name__ == "__main__":
    # 오케스트레이터 에이전트를 생성하고 작업을 요청
    orchestrator = OrchestratorAgent()
    topic = "멀티 에이전트 시스템"
    final_report = orchestrator.run(topic)

    print("--- 최종 결과물 ---")
    print(final_report)
```
