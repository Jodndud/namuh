# React Hooks

### React Hooks란
- 함수형 컴포넌트에서도 상태 관리나 생명주기 같은 기능을 쓸 수 있게 해주는 React의 기능
- 기존에는 이러한 기능을 클래스형 컴포넌트에서만 쓸 수 있었는데, React16.8에 Hooks가 도입되면서 함수형에서도 가능해졌다고 함

### 주요 Hooks
| Hook            | 역할            | 설명                        |
| --------------- | ------------- | ----------------------------- |
| **useState**    | 상태 관리         | 컴포넌트 내에서 데이터를 저장하고 업데이트       |
| **useEffect**   | 생명주기 관리       | 마운트/언마운트/업데이트 시 실행할 코드 지정     |
| **useContext**  | 전역 상태         | Context API로 전역 상태 공유         |
| **useRef**      | DOM 접근 및 값 기억 | DOM 요소나 값에 직접 접근할 때 사용        |
| **useMemo**     | 값 메모이제이션      | 계산 비용이 큰 연산 결과를 저장            |
| **useCallback** | 함수 메모이제이션     | 불필요한 함수 재생성을 방지               |
| **useReducer**  | 복잡한 상태 관리     | Redux처럼 state를 reducer 함수로 관리 |

### 정리
- use로 시작하는 내장 Hook이 있고, 직접 커스텀 Hook을 만들 수도 있음
- 함수형 컴포넌트(단순함)와 클래스형 컴포넌트(기능성)의 장점을 합친 방식
