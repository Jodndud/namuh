## KPT

#### 💪 **Keep (계속할 점)**

- 로봇팔 제어를 어떻게 할지 공부해야함(난이도 극상)
- 프로젝트 목표에 따라 강화학습 도입 검토

#### 😭 **Problem (문제점)**

- 로봇팔의 정확한 용도가 정해져있지 않음

#### 🤙 **Try (새로운 시도)**

- URDF(Unified Robot Description Format)
  로봇의 3D 설계도 + 기구학 정의서(XML)
  ROS에서 로봇의 "몸체 설계도" 역할

##### 기본 구조

```xml
<robot name="my_robot">

  <!-- 1️⃣ 링크 정의 (로봇의 각 부품, rigid body) -->
  <link name="base_link">
    <visual>   <!-- 시각화용 (모양) -->
      <geometry><box size="1 1 0.2"/></geometry>
      <material name="gray"/>
    </visual>
    <collision> <!-- 물리 시뮬레이션용 (충돌) -->
      <geometry><box size="1 1 0.2"/></geometry>
    </collision>
  </link>

  <!-- 2️⃣ 조인트 정의 (부품 간 연결 방식) -->
  <joint name="base_to_arm" type="revolute">
    <parent link="base_link"/>
    <child link="arm_link"/>
    <origin xyz="0 0 0.2" rpy="0 0 0"/>
    <axis xyz="0 0 1"/> <!-- 회전축 -->
    <limit lower="-1.57" upper="1.57" effort="10" velocity="1"/>
  </joint>

  <!-- 3️⃣ 다른 링크 -->
  <link name="arm_link">
    <visual><geometry><cylinder length="0.5" radius="0.05"/></geometry></visual>
  </link>

</robot>
```

##### 주요 구성요소 요약표

| 태그          | 의미                | 예시                               |
| ------------- | ------------------- | ---------------------------------- |
| `<robot>`     | 로봇 전체 루트      | `<robot name="arm_bot">`           |
| `<link>`      | 강체(Body Part)     | `base_link`, `arm_link`, `gripper` |
| `<joint>`     | 링크 간 연결        | `revolute`, `prismatic`, `fixed`   |
| `<origin>`    | 부모-자식 위치 관계 | `xyz="0 0 1" rpy="0 0 0"`          |
| `<axis>`      | 회전/이동 축        | `xyz="0 0 1"`                      |
| `<limit>`     | 관절의 제한         | `upper/lower` 등                   |
| `<visual>`    | RViz용 그래픽 표현  | `<geometry><box/></geometry>`      |
| `<collision>` | Gazebo 충돌 처리용  | `<geometry>` 동일                  |
| `<inertial>`  | 질량·관성 정보      | 물리 시뮬레이션용                  |

##### 조인트 타입 요약

| 타입         | 설명                           | 예시              |
| ------------ | ------------------------------ | ----------------- |
| `fixed`      | 고정 연결 (움직이지 않음)      | 베이스 ↔ 월드     |
| `revolute`   | 회전(θ)                        | 로봇 팔 관절      |
| `continuous` | 360° 회전                      | 바퀴              |
| `prismatic`  | 직선 이동                      | 리니어 액추에이터 |
| `floating`   | 6자유도 (x,y,z,roll,pitch,yaw) | 드론              |
| `planar`     | 평면 내 이동                   | XY 평면 로봇      |
